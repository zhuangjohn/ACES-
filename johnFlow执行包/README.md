# 初始化

## 运行

使用命令行

```
node init.js
```

或点击

```
init.command
```



## 结果

> index.js //任务代码
>
> demoCommand.txt //用于配置启动器，复制到新的command/bat文件中
>
> config
>
> > $work计划任务启动器.js //中间代码，无需更改
> >
> > workflowConfig_default.js //参考配置
> >
> > workflowConfig.js          //配置所需执行的命令及顺序
>
> files //用于存放任务文件
